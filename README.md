# DOCKER COMPOSE INSTRUCTIONS

**TO DOCKER BUILD AND RUN:**

- docker-compose build
- docker-compose up
- docker-compose up -d

**TO STOP:**

- docker-compose down

**TO NORMAL RUN:**

- yarn start

# ESLINT

To check : yarn lint-check
To fix : yarn lint-apply

# PRETTIER

To check : yarn pretty-check
To fix : yarn pretty-apply
